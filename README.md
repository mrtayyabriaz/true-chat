# True Chat

Realtime True-Chat app created by using
`React Js` `Express Js` `Socket-IO` `Typescript`

- [VISIT](https://true-chat.onrender.com/chat) website

## Development

### Setup .env

```js
PORT = "3001";
DOMAIN = "http://localhost:3001";
```

or rename `.env.sample` to `.env`

### Build App

```js
npm run build
```

### Start App

```js
npm run build
```
